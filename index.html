CSS3 Animations

This website will teach you the basics of animation in CSS3

animation-name and @keyframes

Make an element into an animation by giving it a name. To actually animate that element, you do @keyframes (animation name)

Example


            div {
              animation-name: rainbow;
              animation-duration: 2s;
            }
            
            @keyframes {
              0% {
                background-color: red;
              }
              50% {
                background-color: green;
              }
              100% {
                background-color: blue;
              }
            }
